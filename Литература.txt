Сети:

Компьютерные сети. Принципы, технологии, протоколы (Виктор Олифер, Наталья Олифер) - отличная и полная книга по компьютерным сетям, все подробно описывается, написано доступно и интересно, точно бы рекомендовал всем интересующимся, так-же хорошая глава по ИБ с подробной теоретической частью.

Атака сетей на уровне протоколов (Джеймс Форшоу) - Книга дает представление о структуре протоколов и вытекающих из этого уязвимостей, написана достаточно сложно для восприятия, но думаю что оно того стоит.

Рекомендовал бы найти на YouTube видео Кирилла Кринкина (https://www.youtube.com/watch?v=BJSITWkSDQg) отлично объясняет сети и ОС.

SQL:

SQL Полное руководство Третье Издание (Джеймс Грофф, Пол Вайнберг, Эндрю Оппель) - Отличная книга для тех кто хочет разобраться в SQL! Написано все доступным языком, читается на одном дыхании. Подробно разбирается синтаксис и выполнения команд SQL, описаны спецификации SQL.

MySQL по максимуму (Бэрон Шварц, Петр Зайцев, Вадим Ткаченко) - Отличная книга для того что-бы разобраться в профилировании и оптимизации MySQL. Написано доступным языком, читается интересно.

Stepik - Есть отличный курс на патформе Stepik, рекомендовал бы его пройти разработчикам.

Golang:

Golang для профи (Михаил Цукалос) - Хорошая книга с множеством примеров и объяснений, для ознакомления с языком точно бы рекомандовал, минус в том что читается не очень легко, но думаю что это не главное.

Stepik - Отличный курс по Go, прошел его за пару недель, стал уверенно себя чувствовать в языке.

Безопасность:

Этичный хакинг (Дэниел Г. Грэм) - Отличная книга написана доступным языком с подробными примерами, не дает глубокое понимание ИБ, но как погружение в область 100% попадание.

Безопасность контейнеров (Лиз Райс) - Тоже очень хорошая литература, объясняет что такое контейнер и откуда вытекают уязвимости. В каждой главе примеры консольных команд что-бы самому все "Пощупать"

Безопасность веб-приложений (Эндрю Хоффман) - Можно подчерпнуть пару интересных идей, но книга не сильно информативная, даже не знаю кому ее можно порекомендовать.

Информационная безопасность: Защита и нападение (Бирюков А.А.) - Неплохая книга для погружения в область, интересный подход к подаче материала, уязвимости расписаны по модели OSI

Ловушка для багов (Питер Яворски) - Хорошая, легкая для чтения книга, предлагает читателю вместе с автором разобрать реальные репорты о найденых багах

The Browser Hacker's Handook - Отличная книга для того что-бы разобраться в уязвимостях возникающих из-за работы браузера

Есть отличная книга по SQL injection на английском, с примерами на разных языках. К сожалению не могу сейчас вспомнить, есть на ноутбуке но сейчас нету к нему доступа.

Хакинг искусство эксплойта (Джон Эриксон) - Одна из лучших книг по практической ИБ, затрагивает много тем, написано хоть и сложно, но очень захватывающе, рекомендовал бы всем как минимум для общего развития.

если пройтись по сайтам то рекомендую: portswigger.net, hackthebox.com, https://book.hacktricks.xyz/welcome/readme

Остальное:

Внутреннее устройство linux (Брайан Уорд) - Хорошая книга для понимания ОС Linux

Гид по Computer Science (Вильям Спрингер) - Проходится общим взглядом по темам CS, открыла мне глаза не некоторые вещи, точно бы рекомендовал почитать на досуге.

Приемы объектно ориентированного проектирования (Э. Гамма, Р. Хелм, Р. Джонсон, Д. Влиссидес) - Отличная книга для погружение в паттерны проектирования, сначала введение по использованию паттернов для решения задачь, далее разбираем применение паттернов на редакторе документов и в конце подробно описаны паттерны, примеры их применения, плюсы-минусы. Рекомендовал бы всем интересующимся
